## Regole di Codice

- **File leggeri:** ogni file deve essere breve e focalizzato. Se un file supera ~80-100 righe, spezzalo in file più piccoli.
- **Responsabilità singola:** ogni componente/modulo fa una cosa sola.
- **Niente codice superfluo:** no commenti ovvi, no boilerplate inutile, no utility per casi ipotetici.
